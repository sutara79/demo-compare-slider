# スライドショーのライブラリの比較

## Demo
https://sutara79.github.io/demo-compare-slider/

## 条件
- jQuery1.xで動作。
- スライド用画像の縦横比は一定である。
- スライドショーを画面幅いっぱいに表示したい。
- 画面幅に応じてレスポンシブにサイズが変わってほしい。
- 特定のスライドにリンクを設置したい。
- 前後のスライドへ移るためのコントローラがほしい。
